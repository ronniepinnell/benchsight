4. Quick Start

Prerequisites: Python 3.8+ required

Install Dependencies

pip install flask pandas openpyxl

Option A: Run Admin Portal (Recommended)

cd benchsight_merged
python admin_portal.py

# Open http://localhost:5000

Option B: Export Data Directly

# Export all games
python export_all_data.py

# Export specific games
python export_all_data.py --games 18969,18977

Option C: Open HTML Files

# Open in browser
open html/dashboard_static.html
open html/tracker_v16.html

5. Step-by-Step: Tracking a Game

  1. Open the Tracker - Either through admin portal (/tracker) or directly (tracker/tracker_v16.html)
  2. Enter Game ID - Input the game ID in the header field
  3. Load Roster - Import from file or manually add players
  4. Set Up Video - Paste YouTube URL or load local file
  5. Track Events - Fill player slots, select event type, log event
  6. Track Shifts - Update lineup, click "Log Shift"
  7. Export Excel - Click "Export Excel" to save .xlsx file
  8. Copy to Data Folder - Place in data/raw/games/{game_id}/

Keyboard Shortcuts

6. Step-by-Step: Running ETL

Quick Export (Recommended)

# Export all data from all games
python export_all_data.py

# Check output
ls -la data/output/*.csv

Full ETL Pipeline

# Run complete pipeline
python main.py --process-all

# Run specific stage only
python main.py --stage stage       # Load raw data
python main.py --stage intermediate # Transform
python main.py --stage datamart     # Build outputs

7. Step-by-Step: Deploy to Wix

Option A: GitHub Pages (Recommended)

  1. Create GitHub Repo - Name it "benchsight-dashboard"
  2. Upload HTML Files - Copy contents of html/ folder
  3. Enable GitHub Pages - Settings â†’ Pages â†’ Enable from main branch
  4. Get URL - https://yourusername.github.io/benchsight-dashboard/
  5. Embed in Wix - Add HTML iframe element with your URL

Wix Embed Code

<iframe 
    src="https://yourusername.github.io/benchsight-dashboard/dashboard_static.html" 
    width="100%" 
    height="800px" 
    frameborder="0">
</iframe>
Tip: Update data by pushing new files to GitHub. Wix will show the latest version automatically.

9. Troubleshooting

Issue: "No tracking file found"
Fix: Rename file to {game_id}_tracking.xlsx and place in data/raw/games/{game_id}/
Issue: "Module not found" errors
Fix: Run pip install flask pandas openpyxl
Issue: Wix iframe not loading
Fix: Ensure hosted URL uses HTTPS. Check browser console for errors.
Issue: Data not updating
Fix: Re-run python export_all_data.py, push to GitHub, clear browser cache.
