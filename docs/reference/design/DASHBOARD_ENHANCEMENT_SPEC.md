### New Component: `VideoPlayByPlay.tsx`

```typescript
interface VideoPlayByPlayProps {
  events: FactEvents[]
  videoUrl: string | null
  gameId: number
  homeTeam: string
  awayTeam: string
  homeTeamId: string
  awayTeamId: string
  homeTeamData?: TeamData
  awayTeamData?: TeamData
  playersMap?: Map<string, PlayerData>
}

interface VideoSyncState {
  currentTime: number
  isPlaying: boolean
  selectedEventId: string | null
  playbackSpeed: number
}
```

### Features
1. **Video Player Panel** (left side, 60% width)
   - Embedded video player (HTML5 video or iframe for YouTube/Vimeo)
   - Custom controls: play/pause, seek, speed, fullscreen
   - Timeline with event markers (goals = red, penalties = yellow)
   - Picture-in-picture support

2. **Event List Panel** (right side, 40% width)
   - Scrollable event list grouped by period
   - Click event to jump to video timestamp
   - Highlight current event as video plays
   - Same filters as current PlayByPlayTimeline

3. **Sync Behavior**
   - Video time â†’ highlight corresponding event
   - Click event â†’ seek video to `running_video_time`
   - Auto-scroll event list to visible range

### Data Requirements
```sql
-- Ensure fact_events has video timestamps
SELECT
  event_id,
  running_video_time,  -- Seconds from video start
  video_url            -- Optional per-event video URL
FROM fact_events
WHERE game_id = ?
ORDER BY time_start_total_seconds
```

### Implementation Notes
- Use `react-player` or native `<video>` element
- Store video URLs in `dim_schedule.video_url` or `fact_video` table
- Video timestamp calculation: `running_video_time = period_offset + (period_duration - time_remaining)`

   - Grouped by line/position

   - Hover shift â†’ show tooltip with details
   - Click shift â†’ open video at shift start
   - Drag to zoom into time range
   - Period separators

3. **Video Integration**
   - Mini video player in corner or modal
   - Auto-play shift video on selection
   - Show events that occurred during shift

### 9.2 Views for Enhanced Analytics

```sql
-- Player position heatmap data
CREATE VIEW v_player_positions AS
SELECT
  player_id,
  game_id,
  period,
  time_seconds,
  x_coord,
  y_coord,
  zone
FROM fact_tracking
WHERE data_type = 'player_position';

-- Shot density aggregation
CREATE VIEW v_shot_density AS
SELECT
  player_id,
  team_id,
  season_id,
  FLOOR(x_coord / 10) * 10 AS x_bucket,
  FLOOR(y_coord / 10) * 10 AS y_bucket,
  COUNT(*) as shot_count,
  SUM(CASE WHEN is_goal = 1 THEN 1 ELSE 0 END) as goals,
  SUM(xg) as total_xg
FROM fact_shot_xy
GROUP BY player_id, team_id, season_id, x_bucket, y_bucket;
```
