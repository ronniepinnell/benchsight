# Screenshots

**UI/UX inspiration screenshots from hockey analytics platforms**

Last Updated: 2026-01-15

---

## Overview

This directory contains 88+ screenshots collected from various hockey analytics platforms, including NHL.com, ESPN, and specialized analytics sites. These screenshots serve as design inspiration for the BenchSight dashboard.

---

## Screenshot Categories

**Note:** Screenshots are currently in the root of this directory. They can be organized into subdirectories by category:

- `player-pages/` - Player profile page designs
- `game-pages/` - Game detail page layouts
- `team-pages/` - Team profile page designs
- `analytics/` - Analytics dashboard patterns
- `wireframes/` - Generated wireframes (when created)
