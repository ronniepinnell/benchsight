### Query 1: Get All Highlights for a Game with Video Info

```sql
SELECT 
    h.highlight_key,
    h.description,
    h.video_start_time,
    v.video_url,
    vt.video_type_name,
    hc.highlight_category_name,
    hc.icon
FROM fact_highlights h
JOIN fact_video v ON h.video_key = v.video_key
JOIN dim_video_type vt ON v.video_type_id = vt.video_type_id
JOIN dim_highlight_category hc ON h.highlight_category_id = hc.highlight_category_id
WHERE h.game_id = '19045'
ORDER BY h.video_start_time;
```

### Key Takeaways

1. **dim_video_type**: Defines what types of videos exist (Full_Ice, Broadcast, etc.)
2. **fact_video**: Stores actual video data for each game (URLs, timing)
3. **fact_events**: Contains all game events, some marked as highlights
4. **dim_highlight_category**: Defines highlight categories (Goal, Save, Hit, etc.)
5. **fact_highlights**: Links highlighted events to videos with timing information

**The Flow:**
- Videos are loaded from Excel files â†’ `fact_video`
- Events are tracked in games â†’ `fact_events`
- Highlighted events are extracted â†’ `fact_highlights`
- All linked via foreign keys for easy querying
